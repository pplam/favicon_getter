Favicon Getter
==========

A small library that get favicon url of given url

## Installation

  `npm install favicon-getter`

## Usage

~~~
  import fetchFavicon from 'favicon-getter'
  fetchFavicon('http://www.example.com').then(console.log).catch(console.log)
~~~

## Tests

  `npm test`
